## ▶️ Running locally

Clone the project

```bash
    https://github.com/ricardodev10/find-your-duo.git

    git@github.com:ricardodev10/find-your-duo.git
```

Enter the backend directory

```bash
    cd server
```

install the dependencies

```bash
    npm install
```
